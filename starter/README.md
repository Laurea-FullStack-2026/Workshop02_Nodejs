# Starter Folder

## Instructions

This is your workspace for completing the workshop tasks.

### Getting Started

1. Make sure you have Node.js installed:
   ```bash
   node --version
   ```

2. Create your task files in this folder:
   - `task1.js`
   - `task2.js`
   - `task3.js`
   - `task4.js`
   - `task5.js`

3. Initialize your npm project:
   ```bash
   npm init -y
   ```

4. Install required packages when needed:
   ```bash
   npm install <package-name>
   ```

### Running Your Code

To run any JavaScript file:
```bash
node filename.js
```

For example:
```bash
node task1.js
```

### Tips

- Test each task after completing it
- Read error messages carefully
- Use `console.log()` for debugging
- Refer to the [requirement.md](../requirement.md) for detailed task specifications

### Need Help?

- Check the Node.js documentation: https://nodejs.org/docs/latest/api/
- Review the main README.md in the root folder
- Ask your instructor or TA
